QTLocator: Automated QTL analysis

An R-script for the automated mapping of many phenotypes using multiple QTL mapping

Current version 0.1-0 (07 March 2011)
[Home |Download |FAQ |Manual |News/Updates ]

Jump into QTLocator

One time setup

  • Download, install and start R
  • Install the dependencies into R using CRAN:
    • R/qtl
    • VGAM
    • snow
  • Download QTLocator.R

Creating a config file

Start by downloading the example config file and edit the options.

Running an analysis

Start R and source the QTLocator scipt by typing into the R environment:

source("QTLocator.R")

Then start the analysis by typing:

doAnalysisOn("configfile.ini","path_to_configfile")

Results

Results will be generated in the directory specified. Per trait a folder is created with intermediate QTL mapping file and per trait results. In the main directory the summarized output will be stored.
